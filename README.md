# MCP Security Index

Weekly-refreshed leaderboard of public MCP servers scanned with
[`agent-audit-kit`](https://github.com/sattyamjjain/agent-audit-kit).

- Data: `data/index.json`
- Snapshot history: `data/history.json`
- Per-server cards: `server/*.html`

Findings are embargoed 90 days per
[disclosure policy](https://github.com/sattyamjjain/agent-audit-kit/blob/main/docs/disclosure-policy.md).
