**Released:** 2026-04-29

**Rule count:** 188 (was 180)
**Test count:** 898 (was 869)

## Headline

**CrewAI four-CVE exploit chain** (CERT/CC VU#221883, ThaiCERT
2026-04-02) + **AIVSS v0.8 scoring** during the public-review window
(opened 2026-04-16) + **LangChain prompt-loader CVE-2026-34070** +
**Prisma AIRS catalog mapper** + **OpenClaw provisional privesc**
(IronPlate 2026-04-07 CVSS 9.9).

Plus four new product surfaces: `aak score --aivss` (SARIF
annotator), `aak watch-cve` (CVE-feed daemon), `aak coverage
--source prisma-airs`, and `aak rule lint`.

### CLI

- `aak score <sarif> --aivss` — annotate SARIF with AIVSS v0.8
  scores. Polymorphic with the legacy `aak score <path>` letter-grade
  scorer.
- `aak coverage --source prisma-airs --fail-under 60` — coverage
  matrix against the public Prisma AIRS attack catalog.
- `aak watch-cve --feeds ox,cert-cc,thaicert,ironplate
  --interval-seconds 1800 --dry-run` — CVE-feed daemon. Framework
  + state file ship today; per-feed fetchers in v0.3.11 (`pip
  install agent-audit-kit[watch]`).
- `aak rule lint --ci` — validate the RuleDefinition registry
  against AAK metadata invariants. Clean today on all 188 rules.

### Runtime helpers

- `agent_audit_kit.scoring.aivss.score_finding(rule_meta, runtime_ctx)`
  emits an `AIVSSScore` (v0.8) with AARS / environmental / threat /
  exploit-availability vectors.
- `agent_audit_kit.checks.path_under_root(path, root)` — generic
  path-traversal anchor; suppresses
  `AAK-LANGCHAIN-PROMPT-LOADER-PATH-001`.
- `agent_audit_kit.checks.openclaw.assert_role_allowlisted` —
  suppresses `AAK-OPENCLAW-PRIVESC-001`.
- `agent_audit_kit.sanitizers.crewai` — four guards covering the
  CVE-2026-2275 / 2285 / 2286 / 2287 chain.

## Upgrade notes

- No breaking changes. `RULE_COUNT` is now `188`; `__version__` is
  `0.3.10`.
- `aak score` is now polymorphic — pass a directory for the legacy
  letter-grade scorer; pass a SARIF file with `--aivss` for AIVSS
  v0.8 annotation.
- The CVE-feed daemon (`aak watch-cve`) ships with stub fetchers in
  v0.3.10; real OX RSS / NVD JSON / IronPlate fetchers land in
  v0.3.11. The state file lives in `~/.agent-audit-kit/watch-state.json`.
