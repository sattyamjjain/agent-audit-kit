# AgentAuditKit v0.3.6 — release notes

**Released:** 2026-04-26
**Tag:** `v0.3.6`
**Rule count:** 169 (was 161)
**Test count:** 800+ (was 772)
**Predecessor:** [v0.3.5](https://github.com/sattyamjjain/agent-audit-kit/releases/tag/v0.3.5) (2026-04-25)

## Headline

OX MCP STDIO architectural class — Python / TypeScript / Java / Rust SDK
rules + marketplace-fetch detection + Azure / LMDeploy / Splunk variants
+ one-shot `mcp-ox-2026-04` preset.

This release converts AAK's posture from CVE-by-CVE response to
class-coverage. The 8 CVEs aggregated by OX Security's April 2026
reframe (CVE-2026-30615, 30617, 30623, 22252, 22688, 33224, 40933,
6980) all trace to `StdioServerParameters(command=<network_input>)` in
the upstream MCP SDKs; v0.3.6 ships one rule per language plus the
marketplace-fetch single-line shape Cloudflare's MCP-defender writeup
called out as the highest-risk bug in the wild.

### Scanners (4 new + 1 extension)

- `agent_audit_kit/scanners/mcp_stdio_params.py` — Python AST + TS regex
  + Java regex (with nested-paren-safe terminator scan) + Rust regex
  with `use`-import resolution.
- `agent_audit_kit/scanners/mcp_marketplace_fetch.py` — Python AST
  fetch→spawn flow detection + TS regex pair + opt-in trust file.
- `agent_audit_kit/scanners/mcp_server_auth.py` — server-author Azure
  MCP missing auth.
- `agent_audit_kit/scanners/splunk_mcp_config.py` — YAML structured
  pass + `inputs.conf` line-pair regex.
- `agent_audit_kit/scanners/ssrf_redirect.py` — extended with LMDeploy
  VL adapter.

### Preset infrastructure

- `agent_audit_kit/presets/__init__.py` — `load_preset()` + registry
  helpers.
- `agent_audit_kit/presets/mcp-ox-2026-04.yaml` — bundles 12 rules
  covering the OX MCP class.
- CLI: `aak scan --preset mcp-ox-2026-04 .` (unions with `--rules`).
- Action: `preset:` input wired through `entrypoint.sh`.
- See [docs/presets/mcp-ox-2026-04.md](../docs/presets/mcp-ox-2026-04.md).

### Suppression files

- `.aak-mcp-marketplace-trust.yml` — opt-in URL allow-list for
  AAK-MCP-MARKETPLACE-CONFIG-FETCH-001 (requires non-empty
  `justification:` per entry).

## Caveats

- **Rust adapter is regex-only** until #22 lands tree-sitter-rust;
  expect ~10% false-positive rate on macro-heavy codebases.
- **CVE-2026-33626** (LMDeploy VL SSRF) is GHSA-only at cut time. NVD
  enrichment pending; pin floor will tighten in v0.3.7.
